## 👍🏻 구현 기능 상세

### 1. 이메일/소셜 로그인&회원가입&로그아웃

- 카카오 소셜로그인 & 회원가입 (OAuth2.0)
- 이메일 로그인

### 2. 프로젝트 등록

- s3 서버로 정적 파일(이미지) 전송
- 이미지 파일 리사이징
- 이미지 포맷 변환(jpeg)
- 이미지 파일명 생성(UUID)
- 이미지 갱신시, 기존 이미지 파일 삭제
